PDF Report Generator - Student/Company Data

Overview
A professional Python tool for generating PDF reports from student or company data with formatted tables, summaries, and professional styling.

 Features

 Core Features
 Add student data (name, ID, email, department, marks, attendance)
 Add company employee data (name, ID, email, role, performance, salary)
 Load data from CSV/JSON files
 Generate professional PDF reports with:
   Executive summary
   Data tables with proper formatting
   Date/time stamps
 Statistics and summaries
 Automatic file saving with unique filenames
 Data validation and error handling

 Advanced Features
 Professional table styling with alternating row colors
 Summary statistics and calculations
 Modular code structure
 Dynamic support for multiple entries
 Reports directory auto-creation

 Installation

 Prerequisites
 Python 3.7 or higher
 pip package manager

 Setup
1. Clone the repository:

git clone <your-github-repo-url>
